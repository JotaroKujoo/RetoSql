## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/JotaroKujoo/retosql

# Access
$ cd retosql

# Install dependencies
$ npm i

# Run the project
$ npm run dev

# The server will initialize in the <http://localhost:3000>
```
